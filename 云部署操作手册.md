## 第二步：ICP备案（15-20个工作日）

### 2.1 准备材料
```
个人备案需要：
✓ 身份证正反面照片（清晰）
✓ 手机号码（本人实名）
✓ 邮箱地址
✓ 居住地址
✓ 域名证书（阿里云自动生成）

企业备案需要（如果有公司）：
✓ 营业执照
✓ 法人身份证
✓ 公司公章
```

### 2.2 备案流程
```
1. 登录阿里云备案系统
   网址：https://beian.aliyun.com

2. 填写主体信息
   - 选择"个人"
   - 填写身份证信息
   - 上传身份证照片

3. 填写网站信息
   网站名称：MindLink教育平台
   网站内容：教育文化
   服务类型：在线教育
   网站描述：为学生和教师提供AI辅助的教育咨询服务

4. 上传资料
   - 身份证正反面
   - 域名证书
   - 网站备案信息真实性核验单（签字）

5. 提交初审
   - 阿里云审核（1-2个工作日）
   - 如有问题会电话联系

6. 拍照核验
   - 收到短信通知后
   - 使用阿里云APP拍照
   - 或到指定地点拍照

7. 提交管局审核
   - 等待管局审核（10-20个工作日）
   - 审核通过会收到短信和邮件
```

### 2.3 备案注意事项
```
⚠️ 重要提示：
1. 网站名称不能包含"AI"、"智能"等词汇
   ✓ 正确：MindLink教育平台
   ✗ 错误：MindLink AI智能教育平台

2. 个人备案不能涉及商业内容
   - 不能有"购买"、"支付"等功能
   - 不能有广告
   - 纯教育咨询可以

3. 备案期间网站不能访问
   - 需要关闭网站
   - 或使用临时域名

4. 保持手机畅通
   - 管局可能会电话核实
   - 及时接听并配合
```

## 第四步：部署应用（2-3小时）

### 4.1 连接服务器
```bash
# Mac/Linux
ssh root@你的服务器IP

# Windows
# 使用Xshell、PuTTY或Windows Terminal
```

### 4.2 安装基础软件
```bash
# 更新系统
apt update && apt upgrade -y

# 安装Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 安装PostgreSQL
apt install -y postgresql postgresql-contrib

# 安装Nginx
apt install -y nginx

# 安装Git
apt install -y git

# 安装PM2（进程管理）
npm install -g pm2

# 安装Certbot（SSL证书）
apt install -y certbot python3-certbot-nginx
```

### 4.3 配置PostgreSQL
```bash
# 切换到postgres用户
sudo -u postgres psql

# 在PostgreSQL中执行
CREATE DATABASE mindlink_db;
CREATE USER mindlink_user WITH PASSWORD 'your_strong_password';
GRANT ALL PRIVILEGES ON DATABASE mindlink_db TO mindlink_user;
\q

# 退出PostgreSQL
exit
```

### 4.4 部署后端
```bash
# 创建项目目录
mkdir -p /var/www
cd /var/www

# 克隆项目
git clone https://github.com/James-Fountain/MindLink.git
cd MindLink/backend

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
nano .env

# 编辑.env文件，修改以下内容：
# NODE_ENV=production
# PORT=3000
# DB_HOST=localhost
# DB_PORT=5432
# DB_NAME=mindlink_db
# DB_USER=mindlink_user
# DB_PASSWORD=your_strong_password
# JWT_SECRET=your_random_secret_key_here
# CORS_ORIGIN=https://mindlink.cn

# 保存并退出（Ctrl+X, Y, Enter）

# 初始化数据库
psql -U mindlink_user -d mindlink_db -f database/postgresql-schema.sql

# 启动后端服务
pm2 start server.js --name mindlink-backend
pm2 save
pm2 startup
```

### 4.5 部署前端
```bash
# 复制前端文件
cd /var/www/MindLink
cp -r pages css js assets /var/www/html/

# 或创建专门的目录
mkdir -p /var/www/mindlink
cp -r pages css js assets /var/www/mindlink/
```

### 4.6 配置Nginx
```bash
# 创建配置文件
nano /etc/nginx/sites-available/mindlink

# 粘贴以下内容：
server {
    listen 80;
    server_name mindlink.cn www.mindlink.cn;

    root /var/www/mindlink;
    index index.html;

    # 前端路由
    location / {
        try_files $uri $uri/ /pages/index.html;
    }

    # 后端API
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }

    # 静态资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        expires 7d;
        add_header Cache-Control "public, immutable";
    }
}

# 保存并退出

# 启用配置
ln -s /etc/nginx/sites-available/mindlink /etc/nginx/sites-enabled/

# 测试配置
nginx -t

# 重启Nginx
systemctl restart nginx
```

### 4.7 配置HTTPS（备案通过后）
```bash
# 等备案通过后，执行以下命令
certbot --nginx -d mindlink.cn -d www.mindlink.cn

# 按提示操作：
# 1. 输入邮箱
# 2. 同意服务条款
# 3. 选择是否重定向HTTP到HTTPS（选择2，强制HTTPS）

# 测试自动续期
certbot renew --dry-run

# 设置自动续期（添加到crontab）
crontab -e
# 添加以下行：
0 3 * * * certbot renew --quiet
```

## 第七步：监控与维护

### 7.1 查看服务状态
```bash
# 查看PM2进程
pm2 status

# 查看日志
pm2 logs mindlink-backend

# 查看Nginx状态
systemctl status nginx

# 查看PostgreSQL状态
systemctl status postgresql
```

### 7.2 配置自动备份
```bash
# 创建备份脚本
nano /root/backup.sh

# 粘贴以下内容：
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="/backup"
mkdir -p $BACKUP_DIR

# 备份数据库
pg_dump -U mindlink_user mindlink_db > $BACKUP_DIR/db_$DATE.sql
gzip $BACKUP_DIR/db_$DATE.sql

# 备份代码
tar -czf $BACKUP_DIR/code_$DATE.tar.gz /var/www/MindLink

# 删除7天前的备份
find $BACKUP_DIR -name "*.gz" -mtime +7 -delete

# 保存并退出

# 添加执行权限
chmod +x /root/backup.sh

# 添加到定时任务
crontab -e
# 添加：每天凌晨2点备份
0 2 * * * /root/backup.sh
```

### 7.3 配置监控告警
```bash
# 安装监控工具
npm install -g pm2-logrotate
pm2 install pm2-logrotate

# 配置日志轮转
pm2 set pm2-logrotate:max_size 10M
pm2 set pm2-logrotate:retain 7
```

## 常见问题解决

### Q1: 无法连接服务器
```
检查：
1. 安全组是否开放22端口
2. 服务器是否正常运行
3. IP地址是否正确
4. 密码是否正确
```

### Q2: 网站无法访问
```
检查：
1. Nginx是否启动：systemctl status nginx
2. 防火墙是否开放80/443端口
3. DNS解析是否生效：ping mindlink.cn
4. 备案是否通过（未备案域名会被拦截）
```

### Q3: 后端API报错
```
检查：
1. PM2进程是否运行：pm2 status
2. 查看错误日志：pm2 logs
3. 数据库是否连接：psql -U mindlink_user -d mindlink_db
4. 环境变量是否正确：cat backend/.env
```

### Q4: 数据库连接失败
```
检查：
1. PostgreSQL是否启动：systemctl status postgresql
2. 数据库是否存在：psql -U postgres -l
3. 用户权限是否正确
4. .env文件中的数据库配置是否正确
```
