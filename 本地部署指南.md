### 2. 启动前端服务

在项目根目录运行：

```bash

# 或使用 Node.js

```

然后访问: http://localhost:8080

### 3. 测试API

健康检查:
```bash
curl http://localhost:3000/health
```

## API接口

- **认证**: http://localhost:3000/api/auth
  - POST /register - 用户注册
  - POST /login - 用户登录
  - GET /verify - 验证Token
  - POST /logout - 用户退出

- **对话**: http://localhost:3000/api/conversations
  - GET / - 获取对话列表
  - POST / - 创建新对话
  - GET /:id - 获取对话详情
  - POST /:id/messages - 保存消息

- **历史**: http://localhost:3000/api/conversations/history
  - GET / - 获取对话历史
  - GET /recent - 获取最近对话

- **数据库**: http://localhost:3000/api/database
  - GET /stats - 获取数据库统计

## 停止服务

查找并停止后端进程:
```bash
# 查找进程
ps aux | grep "node server.js"

# 停止进程（替换 PID）
kill <PID>
```

## 故障排查

### 后端无法启动
- 检查PostgreSQL服务是否运行: `pg_isready`
- 检查端口3000是否被占用: `lsof -i :3000`

### 前端无法连接后端
- 确认后端服务正在运行
- 检查浏览器控制台的CORS错误
- 确认前端配置文件中的API地址正确
